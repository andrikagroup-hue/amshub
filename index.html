⚠️ Setup diperlukan: Tabel password_vault belum ada. Jalankan SQL ini di Supabase SQL Editor lalu refresh:
CREATE TABLE IF NOT EXISTS password_vault ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), label text NOT NULL, tipe text DEFAULT 'Password', akun text, nilai text NOT NULL, kategori text DEFAULT 'Lainnya', catatan text, is_private boolean DEFAULT true, created_by text, created_at timestamptz DEFAULT now() ); ALTER TABLE password_vault ENABLE ROW LEVEL SECURITY; CREATE POLICY "read_password_vault" ON password_vault FOR SELECT USING (true); CREATE POLICY "manage_password_vault" ON password_vault FOR ALL USING (true) WITH CHECK (true);
🔐 Catatan keamanan: password di sini tersimpan di database perusahaan (tidak terenkripsi tambahan). Jangan simpan password pribadi di luar keperluan kerja, dan gunakan mode Privat untuk akun yang tidak perlu dilihat tim lain.
